# AtomQuest — OKR & Goal Management Platform 🎯

A comprehensive goal-setting and performance management application built for the Atomberg Hackathon. AtomQuest empowers employees, managers, and admins to align on objectives, track progress, and drive results.

## Features

- 🎯 **Goal Lifecycle Management** — Create, submit, approve, reject, and rework goals with a complete workflow
- 📊 **Employee Dashboard** — Track personal goals, progress, and check-ins at a glance
- 👥 **Manager Tools** — Approve/reject goals, review check-ins, and oversee team performance
- 🛡️ **Admin Controls** — Manage users, escalation rules, audit trails, and analytics reports
- 🔔 **Real-time Notifications** — Stay updated on goal status changes and team activities
- 📱 **Responsive Design** — Works seamlessly across desktop, tablet, and mobile

## Tech Stack

- **React 19** + **TypeScript**
- **React Router v7** (framework mode with SSR)
- **CSS Modules** for scoped styling
- **Radix UI** for accessible components
- **Recharts** for data visualization
- **Tabler Icons** for iconography

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
